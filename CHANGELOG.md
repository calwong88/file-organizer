# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-02

### Added
- Initial release of File Organizer
- GUI folder selection using tkinter
- Automatic file categorization by extension
- Support for Images (.jpg, .png)
- Support for PDFs (.pdf)
- Support for Documents (.txt, .docx)
- Support for Videos (.mp4, .mkv)
- Support for Audio (.mp3)
- Support for Books (.epub, .azw3)
- Support for Other files (.zip, .ipynb, .csv)
- Automatic folder creation for categories
- File listing and extension display
- Error handling for file operations
- Console output for operation status

### Known Issues
- No duplicate filename handling
- Does not process subdirectories
- Limited file extension support
